🌙 Vyra — Luna’s Sanctum

Vyra is an AI-powered spiritual assistant designed to provide intuitive guidance, clarity, and insight based on user queries.

It blends structured logic with reflective questioning to help users explore patterns, decisions, and personal direction.

---

✨ Features

- 💬 Conversational AI interface (Streamlit)
- 🧠 Intent detection system
- 🔮 Context-aware response generation
- ⚡ FastAPI backend for scalable architecture
- 🌐 Deployable web interface

---

🛠️ Tech Stack

- Python
- Streamlit (Frontend UI)
- FastAPI (Backend API)
- Requests
- Anthropic API (LLM integration)

---

🚀 How to Run Locally

# Clone the repository
git clone https://github.com/Uma11-dev/Vyra_assistant.git

# Navigate to project
cd Vyra_assistant

# Install dependencies
pip install -r requirements.txt

# Run backend
uvicorn app.main:app --reload

# Run frontend
streamlit run vyra_ui.py
